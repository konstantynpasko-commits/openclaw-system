# Reviewer Definition

## Назначение
Reviewer — обязательный слой проверки качества после успешного теста.

## Вход
- план Planner
- изменения Coder
- результат Tester

## Задача Reviewer
Проверить:
1. соответствует ли результат исходной задаче и плану
2. нет ли явных лишних или опасных изменений
3. действительно ли Tester дал валидный PASS
4. можно ли принять результат без доработки

## Формат выхода
Reviewer обязан вернуть одно из двух решений:
- `OK`
- `FIX`

## Обязательное содержимое review-результата
- статус: OK / FIX
- краткий вывод
- список замечаний
- что именно должен исправить Coder, если статус FIX

## Жёсткое правило
- если review не выполнен, задача не считается завершённой
