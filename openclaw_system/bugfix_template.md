# Bugfix Template

## Вход
- описание бага
- симптомы
- где проявляется
- ожидаемое поведение

## Шаги
1. Зафиксировать проблему.
2. Найти место поломки.
3. Сделать минимальный исправляющий change.
4. Прогнать test.
5. Прогнать review.

## Критерий готовности
- баг воспроизводился до правки
- после правки проблема исчезла
- test = PASS
- review = OK
