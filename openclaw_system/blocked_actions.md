# Blocked Actions

## Запрещено без отдельного явного разрешения
- разрушительное удаление
- необратимые shell-команды
- опасные конфигурационные изменения
- скрытые внешние действия от имени пользователя
- запуск действий вне понятного контекста задачи

## Запрещено как режим работы
- самовольное расширение объёма задачи
- пропуск review/test там, где они обязательны
- подмена фактов догадками при системной фиксации состояния
- маркировка planned-компонента как active без проверки
