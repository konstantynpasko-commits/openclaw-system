# Runtime Smoke Checklist

## Минимальная проверка runtime перед важной работой
1. Проверить PATH.
2. Проверить наличие базовых команд:
   - git
   - node
   - npm
3. Проверить наличие специфичных инструментов, если они нужны:
   - openclaw
   - rg
   - acpx
4. Если команда отсутствует в PATH, решить:
   - использовать абсолютный путь
   - или зафиксировать ограничение и не опираться на неё

## Цель
Эта проверка нужна, чтобы не путать runtime limitation с ошибкой реализации.
