# Feature Template

## Вход
- описание новой функции
- границы задачи
- что не входит в задачу
- критерий результата

## Шаги
1. Сформировать plan.
2. Выделить минимальный delivery slice.
3. Реализовать feature без выхода за scope.
4. Прогнать test.
5. Прогнать review.

## Критерий готовности
- новая функция работает в заявленных границах
- scope не расползся
- test = PASS
- review = OK
