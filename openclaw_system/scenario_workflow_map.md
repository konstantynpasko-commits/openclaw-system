# Scenario Workflow Map

## Общий контур
Любой сценарий всё равно проходит через:
- plan
- code
- test
- review

## Отличие только во входе
- bugfix → от ошибки
- feature → от новой функции
- refactor → от структурного улучшения

## Итог
Шаблоны сценариев не заменяют основной workflow, а уточняют стартовую рамку задачи.
