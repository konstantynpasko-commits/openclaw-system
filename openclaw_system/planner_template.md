# Planner Template

## Назначение
Этот файл задаёт минимальный обязательный выход Planner для Этапа 1.

## Формат плана
Для каждой задачи Planner обязан выдать:

1. **Цель**
   - что именно должно быть получено на выходе

2. **Ограничения**
   - что нельзя ломать
   - какие есть рамки по среде, доступам и рискам

3. **Шаги выполнения**
   - шаг 1
   - шаг 2
   - шаг 3

4. **Критерий готовности**
   - как понять, что задача выполнена

5. **Проверка Tester**
   - какую команду/проверку нужно запустить
   - какой ожидается результат

## Жёсткое правило
- если плана в этом формате нет, задача не передаётся в Coder
