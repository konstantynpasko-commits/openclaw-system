# Model Activation Checklist

## Перед использованием вспомогательной модели проверить
1. Агент реально разрешён в текущем конфиге?
2. Есть ли понятная причина использовать его вместо основного маршрута?
3. Не размывает ли это предсказуемость системы?
4. Нужен ли отдельный лог/фиксация такого использования?

## После использования
- зафиксировать, какой агент использовался
- зафиксировать, была ли польза или аномалия
- при проблемах не переводить этот агент в основной путь автоматически
