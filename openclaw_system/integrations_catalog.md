# Integrations Catalog

## Текущие интеграционные направления
- Telegram
- GitHub
- GitHub Actions
- n8n (целевой слой)
- web search / web fetch
- browser automation

## Текущий статус
### Уже есть в системе
- Telegram как основной интерфейс
- GitHub/GitHub Actions как dev-инфраструктурный слой
- встроенные web-инструменты

### Зафиксировано как целевой слой, но не подтверждено как runtime
- n8n

## Правило
Каждая интеграция должна быть описана в двух измерениях:
1. архитектурная роль
2. фактический runtime-статус
