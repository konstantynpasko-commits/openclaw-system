# Fix Loop Definition

## Назначение
Этот файл фиксирует возвратный цикл после Tester и Reviewer.

## Возврат после Tester
- если Tester вернул FAIL, задача немедленно возвращается в Coder
- Coder исправляет только выявленные проблемы
- после исправления Tester запускается повторно

## Возврат после Reviewer
- если Reviewer вернул FIX, задача возвращается в Coder
- Coder исправляет замечания Reviewer
- после исправления задача снова проходит:
  1. Tester
  2. Reviewer

## Жёсткое правило
- после любого существенного исправления review без повторного test запрещён
- финальный статус возможен только после последовательности PASS → OK
