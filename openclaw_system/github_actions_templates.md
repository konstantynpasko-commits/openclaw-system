# GitHub Actions Templates

## Минимальные шаблоны сценариев

### 1. Validate Structure
- проверяет наличие обязательных файлов
- проверяет базовую целостность структуры проекта

### 2. Run Tests
- запускает проектовые тесты, если они есть

### 3. Run Linters
- запускает линтеры и базовые проверки стиля

### 4. Smoke Check
- запускает минимальную smoke-проверку результата

## Правило использования
Эти шаблоны являются стандартными направлениями для будущего GitHub Actions контура.
На текущем этапе они зафиксированы как часть системной архитектуры, даже если конкретные workflow-yaml ещё не созданы.
